## 🚀 Getting Started

1.  Installation / 安裝
    ```bash
    npm install
    ```
2. Start the development server / 啟動開發服務器
   
   ```bash
   npm run dev
   ```
3. Open in browser / 在瀏覽器中打開
   
   ```bash
   http://localhost:5173
   ```
